Endpoints API

MétodoRutaAuthDescripción
POST/api/jobsBearer JOBS_CLIENT_TOKENEncola un job (shell o http)
GET/api/jobs/:idBearer JOBS_CLIENT_TOKENConsulta estado / resultado
POST/api/runBearer JOBS_CLIENT_TOKENEjecuta comando y espera ≤25 s
POST/api/proxyBearer JOBS_CLIENT_TOKENProxy HTTP saliente vía worker
DEL/api/jobs/archiveBearer JOBS_CLIENT_TOKENVacía todo el store de archivo
DEL/api/jobs/activeBearer JOBS_CLIENT_TOKENVacía todo el store activo

Mantenimiento

Borra los blobs acumulados en jobs-archive y jobs-active. Útil para no acumular peticiones HTTP / shell pasadas.
